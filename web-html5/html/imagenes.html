¿Como añadir una imagen a una pagina web?

Para añadir una imagen a una pagina web, lo primero que tenemos que conocer es la etiqueta para insertarla, esta etiqueta es <img>.
Dentro de esta etiqueta, tiene distintos atributos que iremos viendo para que son cada uno:

Atributo src:

El atributo src es el que se utliza para seleccionar la ubicación donde se encuentra la imagen almacenada. <img src="ubicacion-imagen">

Atributo alt:

Se usa para darle un texto a la imagen si esta no puede cargar por cualqueir motivo y es obligatoria.
<img src="ubicacion-imagen" alt="Descripcion imagen">
